    # 三列布局

    # 方案一:使用 flex 布局

    两边元素宽度固定,中间元素自动拉伸/缩放

    <div class="container">
  <div class="left">左边固定宽度</div>
  <div class="center">
    中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度
  </div>
  <div class="right">右边固定宽度</div>
</div>
<style>
.container {
  display: flex;
  /* 垂直方向上各自按内容填充高度,不然两边元素的高度会跟着中间元素高度一起长 */
  align-items: flex-start;
}

.left {
  width: 200px;
  background-color: #eee;
}

.right {
  width: 150px;
  background-color: #ddd;
}

.center {
  /* 等价于 flex-grow:1;flex-shrink:1,自动伸,缩 */
  flex: 1;
  background-color: cyan;
  margin: 0 10px;
}
</style>

    # 方案二: 浮动布局

    左右两边浮动,中间使用 margin-left,margin-right

    <div class="container">
  <div class="left">左边固定宽度</div>
  <div class="right">右边固定宽度</div>
  <div class="center">
    中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度中间自动填充宽度
  </div>
</div>

<style >
.left {
  width: 300px;
  float: left;
  background-color: #eee;
}

.right {
  float: right;
  width: 250px;
  background-color: #ddd;
}

.center {
  background-color: cyan;
  margin: 0 260px 0 310px;
}
</style>

    有个小缺点:center主体 是最后加载的

    还有一个小缺点: 屏幕宽度足够窄(小于 left_width+right_width)时,right 会被挤掉跑到下一行,发生错乱

    # 方案三: 绝对定位

    .container {
  position: relative;
}
.left {
  position: absolute;
  left: 0;
  top: 0;
  width: 300px;
  background-color: #eee;
}

.right {
  position: absolute;
  right: 0;
  top: 0;
  width: 350px;
  background-color: #ddd;
}

.center {
  background-color: cyan;
  margin: 0 360px 0 310px;
}

    这个方案可以把 center 提到开头先加载, 屏幕宽度小于left_width+right_width时,right 会盖住 left
