    # 动画方案对比

    常见的三种动画方案

    • transform
    • transition
    • animation

    # transform

    包含四个常用的功能：

    • translate：位移
    • scale：缩放
    • rotate：旋转
    • skew：倾斜

    transform 是一种静态的转场,你看到的已经是最终态了

    比如 transform: translateX(50%);,看到的是已经平移完的结果

    配合transition过渡使用,可以看到 过程

    注意的是，transform不支持inline元素，使用前把它变成block

    # transition

    过渡某一个属性,前提是这个属性值有个变化

    比如 width 原来是 100px,后来改成 300px,那么就可以看到过渡效果, 如果值没有变化,也就没有过渡效果

    transition 属性是一个简写属性，用于设置四个过渡属性：

    • transition-property
    • transition-duration
    • transition-timing-function
    • transition-delay

    # animation 自定义动画

    anmiation 常做一些复杂动画,配合 keyframes 定义动画过程里的一些关键帧 from=0%,to=100%

    .text {
  background-color: #fff;
  animation: mix 1.5s linear;
}

@keyframes mix {
  0% {
    background-color: #fff;
  }

  50% {
    background-color: #7b5;
  }

  to {
    background-color: #999;
    transform: translateX(50px);
  }
}
