# dependencies

# 一. prettier

prettier 既有 npm 依赖包版本,也有 IDE 的扩展版本

  1. 安装 IDE 的扩展,那么可以右键 Format Document,建议开启 formatOnSave
  2. 可以在扩展里面进行选项设置,这会应用于所有项目,如果项目中有 prettier 的相关配置文件,会读取该文件进行 reformat(configpath默认读取.prettierrc文件,如果是其他格式的配置文件,需要手动指定文件名)
  3. 安装 prettier 依赖,可以使用脚本去 reformat 项目文件 如prettier --write src/
# 总结
  • IDE 的 prettier 针对所有使用 vscode 打开的项目
  • 通过 npm 安装的 prettier,针对本项目,通过运行脚本使用
