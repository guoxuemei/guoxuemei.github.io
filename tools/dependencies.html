# dependencies

# 一. prettier

prettier 既有 npm 依赖包版本,也有 IDE 的扩展

安装 IDE 的扩展,那么可以右键 Format Document,可以在扩展里面进行相关设置,如果项目有 prettier 的相关配置文件,会读取该文件进行 reformat,建议开启 formatOnSave,

安装 prettier 依赖,可以使用脚本去 reformat 所有项目文件,

  • IDE 的 prettier 针对所有使用 vscode 打开的项目
  • 通过 npm 安装的 prettier,针对本项目,通过运行脚本使用
